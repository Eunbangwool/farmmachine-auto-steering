# RTK 레벨러 — 현장 실측값 종합 체크리스트

> autosteer와 별개. 균평(레벨러) 시스템 전용.
> 확정 방식: **Kubota (7) 레이저 커넥터 신호 모방** (UP/DOWN/HOLD GPIO).
> 트랙터 자체 EH 밸브가 3점 링크를 상하 제어. 비례밸브 직접 구동 안 함.

이 문서의 값들을 채우면 `leveler_core.py`의 `LevelerParams`가 확정됩니다.
대부분은 줄자/멀티미터로 측정, 유압 응답은 캘리브레이션 주행으로 측정합니다.

---

## A. Kubota (7) 레이저 커넥터 — 전기 인터페이스 ★최우선

> 가이드: 멀티미터 + 1kΩ 저항 + 점퍼. 시동 OFF에서 저항, ON에서 전압.
> 트랙터 회로 절대 수정 금지. (7) 커넥터만 사용.

| # | 항목 | 측정 방법 | 기록란 |
|---|------|----------|--------|
| A1 | 핀 수 | 캡 분리 후 육안 (보통 4핀/6핀) | ___ |
| A2 | GND 핀 | 저항 측정, 차체와 0Ω | 핀 ___ |
| A3 | +12V 핀 | 시동 ON, GND 대비 12V | 핀 ___ |
| A4 | UP 입력 핀 | 12V 인가 시 3P 상승 | 핀 ___ |
| A5 | DOWN 입력 핀 | 12V 인가 시 3P 하강 | 핀 ___ |
| A6 | Active 방향 | 12V=동작(High) / GND=동작(Low) | High / Low |
| A7 | +12V 핀 공급 전류 | 포토MOS 구동 가능한지 (수십 mA) | ___ mA |
| A8 | 커넥터 품번 | 캡/하우징 각인 또는 Kubota 부품표 | ___ |

**판정 절차:**
1. 시동 OFF → 모든 핀쌍 저항 측정 → 0Ω 쌍 = GND/공통
2. 시동 ON, 3P 모드=포지션 → 각 핀 전압 → 12V=전원, 0V=GND, 부동=신호입력
3. 3P 모드=레이저 선택 시도 → 안 되면(매뉴얼대로) 신호핀 확인됨
4. 신호핀에 임시 12V 인가 → 레이저 모드 선택 가능 + 3P 상승/하강 확인

## B. 안테나 ↔ 블레이드 기하 — z 계산용 ★가장 중요

> 레벨러는 "블레이드 하단(지면 접촉선) 높이"를 제어. RTK는 안테나 높이를 줌.
> 이 변환이 틀리면 균평 전체가 틀어짐. autosteer의 레버암과 같은 원리, 단 z축.

| # | 항목 | 기호 | 측정 방법 | 기록란 |
|---|------|------|----------|--------|
| B1 | 안테나 위상중심 ↔ 블레이드 하단 수직거리 | h | 줄자, 지면 평지에서 | ___ m |
| B2 | 안테나 ↔ 블레이드 수평거리(전후) | d | 줄자, 측면에서 | ___ m |
| B3 | 안테나 ↔ 블레이드 좌우 오프셋 | ℓ | 보통 0 (중심정렬) | ___ m |
| B4 | 블레이드 폭 | W | 줄자 | ___ m |
| B5 | 블레이드가 안테나보다 앞/뒤 | - | 뒤=일반적 | 앞 / 뒤 |

**중요:** 블레이드는 작업기다. 안테나를 작업기(블레이드) 프레임에 직접 단다면
h, d가 작고 안정적. 트랙터 캐빈에 단다면 3P 링크 각도에 따라 h가 변하므로
**안테나는 블레이드(작업기)에 직접 장착 권장.** 이 경우 IMU도 작업기에.

## C. 유압 응답 특성 — 제어 튜닝용 ★캘리브레이션 주행으로 측정

> leveler_core.py의 calibrate 모드로 자동 측정 가능 (UP/DOWN 펄스 주고 RTK z 변화 기록).
> 트랙터 감도 다이얼 설정을 고정한 뒤 측정할 것.

| # | 항목 | 기호 | 측정 방법 | 기록란 |
|---|------|------|----------|--------|
| C1 | UP 속도 | v_up | UP 2초 인가, Δz/Δt | ___ cm/s |
| C2 | DOWN 속도 | v_down | DOWN 2초 인가, Δz/Δt | ___ cm/s |
| C3 | 응답 지연(dead time) | t_lat | 신호 ON → 움직임 시작까지 | ___ ms |
| C4 | 코스팅(정지 후 추가이동) | coast | 신호 OFF → 멈출 때까지 Δz | ___ cm |
| C5 | 최소 유효 펄스폭 | t_min | 움직임 만드는 최소 ON 시간 | ___ ms |
| C6 | 트랙터 데드밴드(감도다이얼) | - | 다이얼 위치 고정 기록 | ___ |

**캘리브레이션 절차 (leveler_core.py calibrate 사용):**
1. 평지/포장에서 트랙터 정지, 시동 ON, 레이저 모드
2. UP 2초 인가하며 RTK z 로깅 → 기울기 = v_up, 시작지연 = t_lat
3. UP 정지 후 추가 상승량 = coast
4. DOWN 동일 반복
5. 0.1s, 0.2s, 0.3s... 펄스로 최소 유효폭 t_min 탐색

## F. 포장 기준 — 작업 전 측량 (매 포장)

> 이건 트랙터 고정값이 아니라 포장마다 다름. 작업 시작 시 측량.

| # | 항목 | 방법 | 비고 |
|---|------|------|------|
| F1 | 기준점(벤치마크) | RTK로 1점 고정 측정 | 포장 원점 |
| F2 | 포장 경계 코너 | 경계 따라 주행하며 기록 | 작업 영역 |
| F3 | 표면 측량점 | 블레이드 들고 포장 격자 주행 | 평면 피팅용 |
| F4 | 목표 평면 경사 | 보통 0%, 배수용 0.1~0.3% | 사용자 지정 |

**평면 피팅 원리 (leveler_core.py TargetPlane.fit_from_survey):**
- 표면 측량점들의 최소자승 평면 = 흙 이동량 최소화 평면 (laser leveling 표준)
- 이 평면을 목표로 하면 깎기/메우기 총량 최소
- 경사 지정 시: 평균 높이 유지하며 기울기만 부여

## 채우기 우선순위

```
1. ★ B1, B2 (안테나↔블레이드 h, d)     ← 줄자 5분, 가장 중요
2. ★ A1~A6 (커넥터 핀맵 + Active)        ← 멀티미터 1.5시간
3. ★ C1~C5 (유압 응답)                   ← 캘리브레이션 주행 30분
4.   D1, D2 (가동 범위)                   ← 줄자
5.   E1, E2 (IMU 오프셋)                  ← autosteer와 공유
6.   F (포장 측량)                        ← 작업 직전 매번
```

A, B, C만 채우면 실작업 가능. D~F는 안전/정밀 향상.
